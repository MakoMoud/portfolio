PROJECT #01

Project FP

"Project FP [Flappy Bird] started when my passion for game development was beginning, so I picked up unity and decided to create a game."

"At the time, my experience with game development was very little. However, I was eventually able to learn how to create the game through some tutorials on YouTube. I was able to create a scoreboard, and save it to the local files, create a title page and an exit button for the game, add text and images and customise them for the game UI to look much better, and finally add character collisions and a way for the character to get 'destroyed'."
